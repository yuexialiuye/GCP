## cloud router
不需要搭配internal ip即可跟外部連線，提高安全性
````
創建一台虛擬機，HTTP不需要打勾，Network interfaces裡的External IPv4 address選擇None
進入Network Services頁面，選擇左側Cloud NAT
Get Started
Name:default-nat
Network:default
Region:asia-east1(Taiwan)
Cloud Router
Create a route
Name:default-router
Create
Create
虛擬機SSH即可連線
````
